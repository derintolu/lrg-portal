Typography Guidelines

  • Headlines: Use font-stretch: 125% for extended widths with proper variable font support
  • Body Text: Use font-stretch: normal (100%) for optimal readability
  • Hierarchy: Maintain consistent font weights and stretch values
  • Line Height: 1.5x for body text, 1.2x for headlines
  • Letter Spacing: -0.02em for large headlines, -0.01em for subheadings

Logo Usage Guidelines

  • Logo Mark: Use for social media profiles, favicons, and small applications
  • Brand Mark: Primary logo for headers, business cards, and main brand applications
  • Minimum Sizes: Logo mark 20px digital / 0.25" print, Brand mark 120px digital / 1" print
  • Clear Space: Maintain space equal to the height of the circle around both versions
  • Hierarchy: Prefer the full brand mark when space allows, logo mark for constrained spaces
